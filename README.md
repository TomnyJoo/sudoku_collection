# Sudoku Collection

一个功能丰富的Flutter数独游戏集合，包含多种类型的数独游戏和丰富的游戏功能。

## 项目简介

Sudoku Collection是一个跨平台的数独游戏应用，支持多种数独变体，提供流畅的游戏体验和丰富的功能。

### 支持的数独类型

- **标准数独**：经典的9x9数独游戏
- **对角线数独**：额外要求两条对角线上的数字也不重复
- **拼图数独**：不规则区域的数独游戏
- **杀手数独**：通过 cages 和目标数字来解题
- **武士数独**：由5个重叠的数独组成的大型数独
- **窗口数独**：额外的3x3窗口区域要求

## 主要功能

- **多难度级别**：从简单到专家级别的难度设置
- **游戏统计**：记录游戏时间、完成率等统计数据
- **游戏保存**：自动保存游戏进度，支持继续游戏
- **主题切换**：支持浅色和深色主题
- **音效**：游戏过程中的音效反馈
- **提示系统**：提供解题提示
- **自定义游戏**：支持创建自定义数独游戏
- **多语言支持**：支持中英文界面

## 技术架构

- **框架**：Flutter
- **状态管理**：View Model模式
- **游戏生成**：并行生成算法
- **存储**：本地存储游戏进度和统计数据
- **UI**：响应式设计，适配不同屏幕尺寸

## 快速开始

### 环境要求

- Flutter 3.0+ 
- Dart 3.0+ 
- Android Studio 或 VS Code

### 安装步骤

1. 克隆项目
   ```bash
   git clone https://github.com/TomnyJoo/sudoku_collection.git
   cd sudoku_collection
   ```

2. 安装依赖
   ```bash
   flutter pub get
   ```

3. 运行项目
   ```bash
   flutter run
   ```

## 游戏玩法

1. **选择游戏类型**：在主界面选择喜欢的数独类型
2. **选择难度**：根据自己的水平选择难度级别
3. **开始游戏**：点击开始按钮开始游戏
4. **填写数字**：通过屏幕键盘输入数字
5. **使用功能**：
   - 提示：获取解题提示
   - 撤销：撤销上一步操作
   - 清除：清除当前单元格
   - 笔记：使用笔记功能标记可能的数字

## 特色功能

### 智能生成算法
- 使用并行处理技术生成高质量的数独谜题
- 确保每个谜题都有唯一解
- 根据难度级别调整生成策略

### 游戏统计系统
- 记录每个游戏类型的完成时间
- 计算平均解题时间
- 跟踪游戏完成率

### 响应式设计
- 适配手机、平板等不同设备
- 自适应屏幕方向
- 优化触摸交互体验

## 开发指南

### 添加新的数独类型

1. 在 `games/` 目录下创建新的游戏类型文件夹
2. 实现相应的模型、视图和服务
3. 在主界面添加新游戏类型的入口

### 自定义主题

1. 修改 `common/theme/app_colors.dart` 中的颜色定义
2. 在 `common/theme/app_text_styles.dart` 中调整文本样式
3. 通过设置界面切换主题
